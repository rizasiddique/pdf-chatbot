# DocuMind AI 📄🤖

An AI-powered PDF chatbot that allows users to upload PDF documents and ask questions directly from their content.

## 🚀 Live Demo

Frontend:
https://documind-ai-riza.netlify.app/

Backened:
https://pdf-chatbot-o54h.onrender.com 

## ✨ Features

* Upload PDF documents
* Extract and process PDF text
* Ask questions in natural language
* AI-powered answers using Groq LLM
* Fast document retrieval using ChromaDB
* Modern chat-style interface
* Deployed on Netlify and Render

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* FastAPI
* Python

### AI & Database

* Groq API
* ChromaDB

### Deployment

* Netlify
* Render

## 📖 How It Works

1. User uploads a PDF.
2. PDF text is extracted and processed.
3. Text is stored in ChromaDB.
4. User asks a question.
5. Relevant content is retrieved.
6. Groq LLM generates an answer.
7. Answer is displayed in the chat interface.

## 🎯 Use Cases

* Study notes and exam preparation
* Research papers
* Technical documentation
* E-books and reports
* Quick information retrieval from PDFs
